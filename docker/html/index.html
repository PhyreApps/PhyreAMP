Welcome to PhyreAMP

PhyreAMP is a simple LAMP stack that is built using Docker and Docker Compose. It is designed to be a simple and easy to use development environment for PHP developers.

PhyreAMP is built using the following technologies:

  • Linux
  • Apache
  • MySQL
  • PHP
  • Docker
  • Docker Compose
